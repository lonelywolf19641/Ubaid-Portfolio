# Ubaid Portfolio

A modern personal portfolio website built to showcase my skills, background, and projects in a clean and professional way.

## Overview

This portfolio is designed to present who I am as a developer, what I do, and how to contact me. It includes sections for my work, skills, and background, along with a responsive layout for both desktop and mobile devices.

## Features

- Clean and modern UI
- Fully responsive design
- Sticky navigation bar
- Mobile menu support
- Smooth scrolling
- Skills section
- About section
- Contact navigation
- GitHub link integration
- Built with simple and readable structure

## Tech Stack

- HTML5
- CSS3
- Tailwind CSS
- JavaScript
- Font Awesome
- Google Fonts
